# Zarena
🎨 Zarena — a cozy little hub for casual multiplayer party games you can play with friends in seconds. 🖊️ Draw and guess together, 🌍 race to pin spots on the map, and 🏆 climb the leaderboard. No sign-up, no downloads, just a room code and good vibes. ✨ Built with Next.js on Vercel + PartyKit for realtime magic. 🚀
